# 📁 File Organizer Tool (Python Project)

## 🚀 Project Overview
The File Organizer Tool is a Python-based automation project designed to organize files in a selected directory into different folders based on their file types. This project helps users manage cluttered folders efficiently by automatically sorting files into categories such as Images, Documents, Videos, Audio, Archives, and Others.

In daily computer usage, folders like Downloads or Desktop often become messy with mixed file types. This tool solves that problem by scanning the folder and arranging files in a structured and organized way.

---

## 🎯 Objectives
- To automate file organization
- To reduce manual effort in managing files
- To improve productivity and file accessibility
- To learn Python file handling and automation
- To build a real-world useful project

---

## ✨ Features
- 📂 Automatic file categorization
- 🖼️ Separates images, documents, videos, audio, archives, etc.
- 🔍 Scans selected folder
- 📁 Creates folders automatically if not present
- 🔄 Moves files into respective folders
- ⚠️ Handles duplicate file names safely
- 🖥️ Supports GUI (Graphical User Interface)
- ⌨️ Supports CLI (Command Line Interface)
- 📊 Displays summary after completion
- ⚡ Fast and efficient processing

## 🧠 How It Works
1. User selects a folder (GUI or CLI)
2. Program scans all files in the folder
3. Identifies file type using extension
4. Matches file with category
5. Creates category folders if not exist
6. Moves files to respective folders
7. Handles duplicate names by renaming
8. Displays summary of operation

---

## 📂 File Categories
The program classifies files into:

- Images (.jpg, .png, .jpeg, etc.)
- Documents (.pdf, .docx, .txt, etc.)
- Videos (.mp4, .mkv, etc.)
- Audio (.mp3, .wav, etc.)
- Archives (.zip, .rar, etc.)
- Others (unknown types)
